#rolling-music

- hackathon做出的小东西     （2015‘5’23）

- 一个简单的用手机控制电脑做出反应的小游戏。

- 屏幕上会有箭头升起，当箭头到达指定光圈时需要玩家立即反应做出对应的动作，动作越精准得分越高。	

###目前用到的东西
- 利用deviceMotionHandler获取手机端重力传感器有关信息，传输到电脑页面。

- 利用LeanCloud快速编制了后端- -

- 厌倦了电脑方向键来玩音乐游戏么？来翻转你的手吧！

- 将左右旋转来代替了方向键，简单明了！
